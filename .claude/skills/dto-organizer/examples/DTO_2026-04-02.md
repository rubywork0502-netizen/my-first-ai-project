### ChatCompleteRequest（POST /api/v1/chat/complete，BE → AI）

**Request**

| 欄位 | 型別 | 必填 | 說明 |
|------|------|------|------|
| `request_id` | `string` | ✓ | 請求 ID |
| `session_id` | `string` | ✓ | 會話 ID |
| `user_id` | `number` | ✓ | 使用者 ID（≥1） |
| `message` | `string` | ✓ | 使用者訊息 |
| `scope_ids` | `number[]` | | 可信範圍 ID（RAG filter，由 BE 計算） |
| `history_messages` | `ContextMessage[]` | | 歷史訊息 |
| `session_state` | `SessionState` | | 會話狀態 |
| `prev_version` | `number \| null` | | 前次 session_state 版本（≥0） |
| `options` | `object` | | 可擴充欄位，依需求傳入 |

**Response**

| 欄位 | 型別 | 必填 | 說明 |
|------|------|------|------|
| `request_id` | `string` | ✓ | 對應的請求 ID |
| `response` | `string` | ✓ | AI 回覆內容 |
| `session_state` | `SessionState` | | 更新後的會話狀態 |
| `version` | `number` | ✓ | 當前 session_state 版本 |

### SessionUpdateRequest（POST /api/v1/chat/session/update，BE → AI）

**Request**

| 欄位 | 型別 | 必填 | 說明 |
|------|------|------|------|
| `request_id` | `string` | ✓ | 請求 ID |
| `session_id` | `string` | ✓ | 會話 ID |
| `user_id` | `number` | ✓ | 使用者 ID |
| `user_input` | `string` | ✓ | 使用者輸入 |
| `assistant_response` | `string` | ✓ | 助手回覆 |
| `scope_ids` | `number[]` | | 可信範圍 ID |
| `session_state` | `SessionState` | | 會話狀態 |
| `prev_version` | `number` | ✓ | 前次版本（≥0） |
| `options` | `object` | | 可擴充欄位，依需求傳入 |

**Response**

| 欄位 | 型別 | 必填 | 說明 |
|------|------|------|------|
| `request_id` | `string` | ✓ | 對應的請求 ID |
| `success` | `boolean` | ✓ | 是否更新成功 |
| `version` | `number` | ✓ | 更新後的 session_state 版本 |

## 共用型別

### ContextMessage

| 欄位 | 型別 | 必填 | 說明 |
|------|------|------|------|
| `role` | `"user" \| "assistant"` | ✓ | 訊息角色 |
| `content` | `string` | ✓ | 訊息內容 |

### SessionState

| 欄位 | 型別 | 必填 | 說明 |
|------|------|------|------|
| `topic` | `string` | | 當前對話主題 |
| `context` | `object` | | 會話上下文資料 |
| `version` | `number` | ✓ | 狀態版本號 |
